# PHP Blog Project

php mysql ကိုသုံး၍ ရေးသားသော blog site လေးတစ်ခုဖြစ်ပါတယ်ခင်ဗျ။ adminpanel တွင် user create လုပ်ချင်း,blog post များတင် ခြင်းနှငိ့ user ဘက်တွင် post များကြည့်ခြင်း comments ပေးခြင်းတို့ပြုလုပ်နိုင်ပါတယ်ခင်ဗျ။  login page တွင် admin@gmail.com ,password-admin ဟုရိုက်၍ လည်းကောင်း user@gmail.com , password-user ဟုရိုက်၍လည်းကောင်း admin panel နှင့် user panel ကိုဝင်ရောက်နိုင်ပြီး မိမိစိတ်ကြိုက် register လုပ်ပြီးတော့လည်း ဝင်ရောက် လေ့လာနိုင်ပါတယ်ခင်ဗျ။

- **Live URL:** [https://myblog388.infinityfreeapp.com](https://myblog388.infinityfreeapp.com)
- **Tech Stack:** PHP, MySQL, Bootstrap template
